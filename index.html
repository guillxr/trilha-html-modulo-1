História dos computadores

Os primeiros computadores começaram a surgir nos anos 40. A 1ª geração foi composta pelos computadores Havard Mark I (1944), Colossus (1944) e ENIAC (1945), que eram enormes devido à tecnologia da época. Com a criação dos transistores, surgiu a 2ª geração (1959 a 1965), que ajudou drasticamente a reduzir o tamanho dos computadores. Na 3ª geração (1965 a 1970), foram criados os circuitos integrados, que incluíam transistores, capacitores e resistores. Hoje em dia, estamos na 4ª geração com os computadores modernos e avançando para a 5ª geração.

O que são Clients?

Client é tudo aquilo que está do lado do "cliente", ou seja, o dispositivo que consome um serviço, seja um site, aplicativo ou API. O client utiliza a internet ou um host local para fazer solicitações ao Servidor e também para receber informações. Dessa forma, ele traduz visualmente as informações recebidas do servidor para o dispositivo do "cliente". No geral, o client pode ser um navegador, um dispositivo móvel, ou até mesmo uma TV.

O que são Servers?

Server ou Servidor é responsável por armazenar, processar e distribuir informações, arquivos, serviços, etc. Como mencionado no conceito de Client, o servidor desempenha o papel de enviar e armazenar informações ou arquivos solicitados pelo Cliente. Cada servidor desempenha funções diferentes dependendo do serviço requisitado. Alguns tipos de servidores são: Servidor Web, servidor de E-mail, servidor de Arquivos, servidor firewall, Servidor de Streaming, etc.

Como é uma estrutura básica de um HTML?

A estrutura de um documento HTML é inicialmente formada pela tag <html>, que abre o elemento HTML. Em seguida, temos a tag <head>, que contém os metadados e informações sobre o documento HTML. Esta seção é destinada a informações que não são diretamente exibidas para os usuários finais (Clients), mas são importantes para que o navegador ou outros dispositivos possam processar o documento.

Dentro do elemento <head>, existem diversas tags, como a tag <title>, que é responsável por definir o título da página, exibido na guia do navegador. Após ela, temos o <body>, que é o corpo principal do HTML, onde todo o conteúdo visível da página é colocado. Ele contém todos os elementos que são exibidos para os usuários finais, como textos, imagens, links, vídeos, formulários, etc.

Dentro do <body>, podemos utilizar várias tags, como:

  • Tags de Títulos: <h1> - <h6> para cabeçalhos de diferentes níveis.
  • Parágrafos: <p> para parágrafos de texto.
  • Links: <a> para criar links para outras páginas ou recursos.
  • Imagens: <img> para exibir imagens na página.
  • Listas: <ul> para listas não ordenadas (com marcadores), <ol> para listas ordenadas (numeradas) e <li> que são os itens da lista.
Essas são apenas algumas das muitas tags que podem ser utilizadas dentro do <body> para estruturar e apresentar o conteúdo de uma página HTML.

Entendendo a diferença entre listas ordenadas e não ordenadas

Para explicar a diferença entre as listas ordenadas (ol) e as não ordenadas (ul), vamos começar com a lista ordenada. Elas são listas nas quais os itens são ordenados de acordo com uma sequência numérica ou alfabética, onde são automaticamente numerados em uma sequência crescente (1, 2, 3...) ou em ordem alfabética (a, b, c...), como podemos ver no exemplo a seguir:

  1. Item A
  2. Item B
  3. Item C
Já no caso da lista não ordenada, elas são listas nas quais não há uma ordem específica, ou seja, os itens não fazem parte de uma sequência. Os itens são marcados com símbolos padrão, geralmente círculos ou quadrados, como podemos ver no exemplo a seguir:
  • Item 1
  • Item 2
  • Item 3
